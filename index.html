DX Website Operation Support

Use this form for operational support of DX Platform–managed public websites (break/fix, minor updates, redirects, decommission). Do not use for new website build and hosting, new site launches, migrations, rebuilds, or other demand intake.

Website issue (site is up)
Use when the site is available but something is wrong (broken page, errors on a path, unexpected behavior, minor operational change). Provide impacted URL(s), what you expected vs what happened, and when it started.

Outage / site down (Incident)
Use only when the production site is down or unusable for most users. Provide impacted URL(s), start time, error message/screenshot, and scope (all pages vs specific paths).

Domain redirect
Use for vanity/301 redirects. Provide Source URL and Target URL, and desired effective date/time (if applicable).

Decommission (Site Owner only)
Use to retire a site/domain so it no longer serves content (optionally with redirect). Only the business Site Owner may request. Select the Site Owner (FTE only) and confirm Site Owner attestation.

Other (DX website ops)
Use for operational support that does not fit the above for an existing DX-managed site. Provide a short label plus details and impacted URL(s).
